  // construct/copy/destruct
  io_pool_executor();
};

Description

A io_pool_executor exposes the predefined HPX IO thread pool through an executor interface.

Note

All tasks executed by one of these executors will run on one of the OS-threads dedicated for the IO thread pool. The tasks will not run a HPX-threads.
