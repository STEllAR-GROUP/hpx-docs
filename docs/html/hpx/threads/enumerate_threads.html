Function enumerate_threads

hpx::threads::enumerate_threads

Synopsis

// In header: <hpx/runtime/threads/thread_data_fwd.hpp>


bool enumerate_threads(util::function_nonser< bool(thread_id_type)> const & f, 
                       thread_state_enum state = unknown);

Description

The function enumerate_threads will invoke the given function f for each thread with a matching thread state.

Parameters:

f

[in] The function which should be called for each matching thread. Returning 'false' from this function will stop the enumeration process.

state

[in] This specifies the thread-state for which the threads should be enumerated.
