Function template restore_checkpoint

hpx::util::restore_checkpoint

Synopsis

// In header: <hpx/util/checkpoint.hpp>


template<typename T, typename... Ts> 
  void restore_checkpoint(checkpoint const & c, T & t, Ts &... ts);

Description

Resurrect

Restore_checkpoint takes a checkpoint object as a first argument and the containers which will be filled from the byte stream (in the same order as they were placed in save_checkpoint).

Parameters:

c

The checkpoint to restore.

t

A container to restore.

ts

Other containers to restore Containers must be in the same order that they were inserted into the checkpoint.

Template Parameters:

T

A container to restore.

Ts

Other containers to restore. Containers must be in the same order that they were inserted into the checkpoint.

Returns:

Restore_checkpoint returns void.
